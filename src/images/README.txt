This directory contains images used in the website.

Required images:
- rocket-team.jpg: A high-quality image of the MIT Rocket Team or a rocket launch
  Recommended size: 1200x800 pixels
  Format: JPEG
  File size: Optimize for web (under 500KB)

To add your own image:
1. Place your image file named 'rocket-team.jpg' in this directory
2. Make sure the image is optimized for web use
3. The image should be in JPEG format
4. Recommended dimensions are 1200x800 pixels

Note: The current website uses a placeholder image. Replace it with your actual rocket team image for the best presentation.
